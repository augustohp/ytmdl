# ytmdl

> [YouTube Music] downloader.

[YouTube Music]: https://music.youtube.com

## Overview

Downloads albums from [YouTube Music], setting proper ID3 tags on MP3
files.

## Usage

```sh
ytmdl <url>
```

## Example

Example downloading the "Released and Unreleased, Vol. 2" album by
Funkmaster Ozone.

```sh
ytmdl 'https://music.youtube.com/playlist?list=OLAK5uy_ktS0AruyXOuAZBD5jve3OtBDdCdWpUiRU'
```
